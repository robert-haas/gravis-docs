convertÂ¶

This is the module gravis.convert.

It provides functions to convert graph objects from external libraries to gJGF. Additionally, it contains functions to convert image files or image URLs to self-contained data URLs and HTML elements. This allows to generate visualizations with images inside nodes that do not rely on any external resources.
